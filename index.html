Descripción General

El código implementa un juego de tres en raya en Python utilizando la biblioteca tkinter para la interfaz gráfica. Dos jugadores alternan turnos en un tablero de 3x3, con el objetivo de alinear tres de sus símbolos. La lógica del juego está separada en la clase JuegoGato, que maneja el estado del juego, los movimientos y la detección de ganadores. Los jugadores interactúan con la interfaz gráfica, haciendo clic en los botones que representan las celdas del tablero.

Clase JuegoGato

La clase JuegoGato representa el juego de Tres en Raya, gestionando el estado del tablero, los jugadores, los turnos y la lógica para determinar ganadores y movimientos. Esta clase contiene métodos para inicializar el juego, realizar movimientos, comprobar si hay un ganador y reiniciar el juego. Facilita la interacción entre los jugadores y el tablero, asegurando que las reglas del juego se respeten.

Atributos de la clase
  • tablero:
    • Tipo: String[3][3]
    • Descripción: Matriz que representa el estado del juego. Cada celda puede contener:
      • " " para vacío
      • "X" para el jugador 'X'
      • "O" para el jugador 'O'
  • jugadores:
    • Tipo: String[]
    • Descripción: Array que almacena los símbolos de los dos jugadores.
  • turno:
    • Tipo: int
    • Descripción: Índice que indica de quién es el turno actual (0 para 'X', 1 para 'O').
  • ganador:
    • Tipo: String (o null)
    • Descripción: Almacena el valor del jugador ganador ("X" o "O") o null si no hay ganador.
Métodos de la clase
  • void __init__()
    • Descripción: Inicializa el juego, configurando el tablero a su estado inicial (todas las celdas vacías), reiniciando los jugadores y el turno.
  • int, list comprobar_ganador()
    • Descripción: Verifica las combinaciones posibles en el tablero para determinar si hay un ganador. Retorna un array:
      • None si no hay ganador
      • La combinación ganadora si hay un ganador.
  • Boolean realizar_movimiento(int fila, int col)
    • Descripción: Intenta realizar un movimiento en la posición especificada del tablero.
    • Parámetros:
      • int fila: la fila donde se quiere realizar el movimiento.
      • int col: la columna donde se quiere realizar el movimiento.
    • Retorno: true si el movimiento fue exitoso (la celda estaba vacía y se realizó el movimiento), false si no se pudo realizar (la celda ya estaba ocupada).
  • void reiniciar_juego()
    • Descripción: Restablece el estado del juego a su configuración inicial, permitiendo comenzar una nueva partida.
Tipos de valores en los métodos
  • __init__(): void
  • comprobar_ganador(): (String, List)
  • realizar_movimiento(fila, col): boolean
  • reiniciar_juego(): void
Interfaz

Si la clase JuegoGato implementa una interfaz, podría incluir métodos como:

  • void actualizar_tablero()
    • Descripción: Actualiza el texto de los botones en la interfaz gráfica para reflejar el estado actual del tablero, mostrando los símbolos de los jugadores en las posiciones correspondientes.
  • Void realizar_movimiento(fila, col)
    • Descripción: Intenta realizar un movimiento en la posición especificada del tablero. Retorna true si el movimiento fue exitoso (la celda estaba vacía) o false si no se pudo realizar (la celda ya estaba ocupada).
  • void reiniciar_juego()
    • Descripción: Reinicia el estado del juego, restableciendo el tablero a un estado vacío, configurando el turno al jugador 'X' y estableciendo que no hay ganador.
  • void resaltar_ganador(combinacion)
    • Descripción: Resalta la combinación ganadora en el tablero cambiando el color de fondo de los botones a amarillo.
  • void deshabilitar_botones()
    • Descripción: Deshabilita todos los botones del tablero para evitar movimientos adicionales una vez que el juego ha terminado.
  • void reiniciar()
    • Descripción: Reinicia el juego, restableciendo el estado del tablero y configurando el turno al jugador 'X'. También habilita los botones para permitir nuevas jugadas.
